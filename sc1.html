基礎打底學習計畫

第 1 週：HTML & CSS 基礎

學習內容：

  • HTML 基本標籤：div、p、img、a、form
  • CSS 基礎：選擇器、盒模型、顏色、字體、邊框等
  • 練習製作簡單的靜態網頁，包含文本、圖片、按鈕等基本元素

專案練習：

  • 仿製一個簡單的個人介紹網頁

第 2 週：CSS 排版與佈局

學習內容：

  • CSS Flexbox
  • CSS Grid
  • 實作網頁排版布局（水平垂直居中、導航條、卡片式布局等）

專案練習：

  • 完成個人介紹網頁的排版（使用 Flexbox 和 Grid）

第 3 週：JavaScript 基礎

學習內容：

  • 變數：let、const、var
  • 資料型別：字串、數字、布林、陣列、物件
  • 運算符、函式、條件語句 (if/else)、迴圈 (for, while)

專案練習：

  • 製作一個簡單的計算器或數字猜謎遊戲
